   # English language
    How to config.json
1. If the 2nd child has grown up and has not changed according to the mod, Please be assured that you have set 
---> Stardew Valley\Mods\ChildToNPC\config.json  "AgeWhenKidsAreModified" from  83  to 56﻿﻿  But if you change but don't change, Try entering the game(save) again.
2. You can choose a characters and portraits for each child by going to config.json(by notepad) and changing   "Auto"   from "True"  to  "False"
 After that, it can be changed from the  "default"  to another name, such as  "Abigail" .
3. You can set the curfew for your children in the folder of ChildToNPC at config.json the original value is 1900 (1900 meaning 7pm.)
4. There are 2 types of dresses, apron style and no apron style. If want no apron, change   "Apron"  from   "True"   to  "False".

**bonus** If you set your favorite things to be strange, then prepare yourself.  *giggle* *giggle*

---> Stardew Valley\Mods\ChildToNPC\config.json "AgeWhenKidsAreModified" จาก  83   เปลี่ยนเป็น  56    หากลองทำตามแล้วแต่ยังไม่เปลี่ยน ให้ลองออกและเข้าเซฟใหม่

3. สามารถเปลี่ยนเวลาที่ลูกจะกลับบ้านได้ที่โฟลเดอร์ ChildToNPC  ที่  config.json เวลาปกติอยู่ที่ 1900 (1900 หมายถึง 1 ทุ่ม)
4. ชุดลูกมี 2 แบบคือแบบใส่เอี๊ยมและแบบไม่ใส่ หากไม่ใส่ ให้เปลี่ยน   "Apron"  จาก "True"  เป็น  "False"
